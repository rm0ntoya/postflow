# Modelos de Slides para Carrossel

Catálogo de 35 layouts de slides — posição de imagem, texto, hierarquia tipográfica, uso ideal e mood.

Cada modelo usa o canvas de **1080 × 1350 px** (formato Instagram Stories/Carrossel).

---

---

│  ░░░░░░░░░░░░░░░░░░░░░  │ ← Imagem full-bleed com overlay escuro

│   │  TÍTULO GRANDE  │   │ ← TheBoldFont, 120–160px, branco
│   │  EM CAIXA ALTA  │   │
